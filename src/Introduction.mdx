## What this is

This is a personal, portfolio-safe component library. Every component is a **clean,
generic, reusable building block** — written from first principles, with no ties to any
app, brand, or employer codebase. That's what lets each one be previewed, themed, and
dropped into any project in isolation.

Use the sidebar to browse components. Every page is interactive: tweak props live with
the **Controls** panel, read the auto-generated API docs, and check accessibility with
the built-in **a11y** addon.

## Stack

- **Vue 3** with `<script setup lang="ts">` and the Composition API
- **TypeScript** throughout
- **Tailwind CSS v4** (config-less; tokens live in `src/style.css`)
- **Storybook 10** with the docs, a11y, vitest, and chromatic addons
- **Vite** for builds
